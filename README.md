# Scratch-Dark-Theme

Scratch Dark Theme gives the Scratch 3.0 user interface a refresh with a darktheme and new modern look.

Supported pages so far:
  * scratch.mit.edu
  * scratch.mit.edu/messages
  * scratch.mit.edu/explore
___
